# PixiJS Isometric Demo

Simple isometric tile map using PixiJS with a spritesheet-driven character and WASD movement.

## Features
- Isometric tile map built from a spritesheet.
- Tile collisions using a simple grid.
- Animated character from a spritesheet.
- WASD movement with collision checks.

## Controls
- W/A/S/D: Move the character.

## Run
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build: `npm run build`

## Assets
The spritesheets in public/assets are placeholders. Replace the SVG files with your own spritesheets if desired.
